# Cancer Awareness & Support

A responsive web page built to raise awareness about cancer prevention, early detection, and support resources.

## Features

- **Landing Page**: Clean hero section with awareness message and banner image
- **Contact Form**: Simple contact form (Name, Email, Message) with client-side validation
- **Inspirational Quotes**: Real-time motivational quotes fetched from public API with refresh functionality
- **Responsive Design**: Mobile-friendly layout that works on all devices
- **No Backend Required**: Pure HTML, CSS, and JavaScript implementation

## Tech Stack

- HTML5
- CSS3 (with Flexbox and responsive media queries)
- Vanilla JavaScript (ES6+)
- ZenQuotes API for inspirational quotes
